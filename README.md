# Elzas Math Academy

A web application for Elzas Math Academy, designed to present courses, engage students, and support mathematics learning for CBSE students (Classes 9–12).

## Overview

Elzas Math Academy focuses on:
- Delivering structured mathematics learning
- Supporting CBSE students (Grades 9–12)
- Bridging offline and online teaching
- Improving conceptual understanding through guided practice

## Features

- Academy landing page
- Course and batch information
- Responsive design for mobile and desktop
- Ready for integration with online classes
- Promotional and informational content support

## Tech Stack

- Frontend: React / HTML / CSS / JavaScript
- Backend: (if applicable) .NET / Node.js
- Hosting: Vercel
- Version Control: Git and GitHub

## Installation and Setup

1. Clone the repository

git clone https://github.com/celexv/ElzasMathAcademy.git
cd ElzasMathAcademy

2. Install dependencies

npm install

3. Run the application

npm start

4. Open in browser

http://localhost:3000

## Deployment

The project is deployed using Vercel. Any changes pushed to the repository are automatically deployed.
